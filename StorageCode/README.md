> LAU 是一款基于layui的后台前端框架，遵循layui的模块化开发，是一款简单易上手的后台前端模板

LAU的一些基本特性：

* 使用原生layui开发
* 遵循layui的模块化开发
* 全局iframe，适合处理不好js全局污染的用户（作为一个后端，单页应用经常处理不好js重复加载和全局污染）
* 模仿layuiAdmin，样式美观（事实上是部分抄袭了layuiAdmin的样式）
* 和最新版layui同步更新
